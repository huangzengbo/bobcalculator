Bob's Number Calculator

This is a calculator to calculate type of number.

How to use calculator:

  1. Input an integer
  2. Click "Generate Number" button
  3. See results in result panel

How to run unit test (QUnit):

  1. Open Web Dev Tool or Press "F12"
  2. Go to "Console" panel
  3. Enter "App.calculator.testInit()"
  4. Or click here
